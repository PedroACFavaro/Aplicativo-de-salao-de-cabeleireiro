================================================================================
              GUIA DE USO — SISTEMA DE AGENDAMENTOS
================================================================================

Este documento descreve como utilizar a plataforma de gestão de agendamentos
para salões de cabeleireiro. Leia antes de começar a usar o sistema.

--------------------------------------------------------------------------------
O QUE É O SISTEMA
--------------------------------------------------------------------------------

Plataforma web de gestão para salões de beleza. Por meio dela é possível:

  • Visualizar a agenda de agendamentos no dia ou no mês
  • Criar, editar e cancelar agendamentos
  • Gerenciar a lista de clientes
  • Cadastrar e manter o catálogo de serviços
  • Administrar os dados dos funcionários
  • Enviar mensagens de confirmação, lembrete e cancelamento via WhatsApp

O acesso é feito pelo Google — basta entrar com a conta Gmail cadastrada no
sistema. Não é necessário criar senha separada.

--------------------------------------------------------------------------------
PERFIS DE ACESSO: ADMIN E FUNCIONÁRIO
--------------------------------------------------------------------------------

Existem dois tipos de perfis na plataforma:

  ★ ADMIN
    Acesso completo ao sistema. Pode:
    - Ver e gerenciar todos os agendamentos
    - Criar, editar e excluir agendamentos, clientes, serviços e funcionários
    - Enviar mensagens via WhatsApp para os clientes
    - Configurar durações e preços dos serviços
    - Controlar quem tem acesso à plataforma

  ★ FUNCIONÁRIO
    Acesso limitado. Pode:
    - Entrar na plataforma com a conta Google cadastrada
    - Visualizar somente os próprios agendamentos no calendário
    - Não vê dados de clientes, outros serviços ou outros funcionários

  IMPORTANTE: Um funcionário só consegue acessar a plataforma se o e-mail
  cadastrado na aba "Funcionários" for exatamente o mesmo e-mail da conta
  Google usada para entrar. Se o e-mail estiver errado ou desatualizado,
  o acesso será negado.

--------------------------------------------------------------------------------
CONFIGURAÇÃO INICIAL — O QUE FAZER ANTES DE COMEÇAR A USAR
--------------------------------------------------------------------------------

Antes de usar a plataforma no dia a dia, o admin responsável precisa realizar
duas configurações essenciais:


  ╔══════════════════════════════════════════════════════════╗
  ║  PASSO 1: CONFIGURAR OS SERVIÇOS                         ║
  ╚══════════════════════════════════════════════════════════╝

  Acesse a aba "Serviços" no menu lateral (visível apenas para admins).

  Para cada serviço, verifique e preencha:

    • Nome do serviço (ex: Corte Feminino, Manicure, Escova Progressiva...)
    • Duração em minutos — MUITO IMPORTANTE. A duração define o tempo que o
      agendamento ocupa na agenda. Se estiver incorreta, a visualização do
      calendário não refletirá a realidade e podem ocorrer conflitos de horário.
    • Preço em reais (R$)

  Exemplo:
    Serviço: Escova Simples | Duração: 60 minutos | Preço: R$ 80,00

  Se algum serviço não estiver cadastrado, clique em "Novo Serviço" e preencha
  os dados. Revise todos antes de começar a criar agendamentos reais.


  ╔══════════════════════════════════════════════════════════╗
  ║  PASSO 2: CADASTRAR OS E-MAILS DE TODOS OS FUNCIONÁRIOS  ║
  ╚══════════════════════════════════════════════════════════╝

  Acesse a aba "Funcionários" no menu lateral (visível apenas para admins).

  Para cada funcionário, certifique-se de que o campo E-mail está preenchido
  com o endereço Gmail correto e atualizado.

  Como fazer:
    1. Colete o Gmail de cada funcionário
    2. Na aba "Funcionários", localize o cadastro de cada um
    3. Clique no botão de editar (ícone de lápis) e atualize o campo E-mail
    4. Confirme e salve

  ⚠ ATENÇÃO — APENAS GMAIL É ACEITO:
    O sistema usa autenticação Google. Funcionários com Hotmail, Outlook,
    Yahoo ou outro provedor NÃO conseguirão entrar. Quem não tiver Gmail
    precisa criar uma conta gratuita em gmail.com antes de ser cadastrado.

  ⚠ ATENÇÃO — E-MAIL DEVE SER IDÊNTICO:
    O e-mail cadastrado precisa ser exatamente o mesmo da conta Google usada
    para entrar. Um erro de digitação impedirá o acesso.

  Sobre os perfis:
    - Quem precisa gerenciar agendamentos e clientes → perfil ADMIN
    - Quem precisa apenas ver a própria agenda → perfil FUNCIONÁRIO

--------------------------------------------------------------------------------
COMO ACESSAR A PLATAFORMA
--------------------------------------------------------------------------------

Com o e-mail corretamente cadastrado, o funcionário acessa a plataforma assim:

  1. Abrir o link da plataforma no navegador (Chrome recomendado)
  2. Clicar em "Entrar com Google"
  3. Selecionar ou digitar o e-mail Gmail cadastrado no sistema
  4. O calendário carrega automaticamente com os agendamentos do dia

Não é necessário instalar nenhum aplicativo. A plataforma funciona direto
no navegador, inclusive pelo celular.

--------------------------------------------------------------------------------
O BOTÃO WHATSAPP
--------------------------------------------------------------------------------

O sistema possui um botão de WhatsApp que permite enviar mensagens rápidas
para os clientes diretamente da plataforma. Ao clicar nele, é possível
escolher entre três modelos de mensagem:

  • Confirmação de agendamento
  • Lembrete (para o dia seguinte)
  • Aviso de cancelamento ou remarcação

O texto vem pré-preenchido com os dados do agendamento. É possível revisar
e editar antes de enviar. Ao confirmar, o WhatsApp Web abre automaticamente
com a mensagem pronta.

⚠ IMPORTANTE: O botão de WhatsApp só aparece para clientes que possuem
número de telefone cadastrado na aba "Clientes". Ao cadastrar novos clientes,
sempre inclua o número de celular com DDD (ex: 11912345678).

--------------------------------------------------------------------------------
DICAS PARA O DIA A DIA
--------------------------------------------------------------------------------

  ✔ ATUALIZAÇÃO AUTOMÁTICA
    O calendário se atualiza automaticamente a cada 3 minutos. Para forçar
    uma atualização imediata, clique no botão ↻ no topo da tela.

  ✔ CONFLITOS DE HORÁRIO
    Se dois agendamentos do mesmo funcionário se sobrepuserem, o sistema
    exibirá um aviso de conflito no topo do calendário.

  ✔ VISÃO DIA E MÊS
    É possível alternar entre a visão do dia (grade horária detalhada) e a
    visão do mês (visão geral).

  ✔ BUSCA NAS TABELAS
    Nas abas de Agendamentos, Clientes, Serviços e Funcionários há uma barra
    de pesquisa para encontrar registros rapidamente.

  ✔ OBSERVAÇÕES NOS AGENDAMENTOS
    Ao criar ou editar um agendamento, o campo Observações pode ser usado para
    registrar informações como preferências, alergias a produtos ou combinações
    especiais.

--------------------------------------------------------------------------------
CHECKLIST DE CONFIGURAÇÃO INICIAL
--------------------------------------------------------------------------------

Antes de usar a plataforma no dia a dia, confirme que foi feito:

  [ ] Revisar todos os serviços cadastrados e garantir que a DURAÇÃO em minutos
      está correta para cada um

  [ ] Coletar o Gmail de todos os funcionários

  [ ] Verificar se cada funcionário já tem um Gmail (quem não tiver precisa criar)

  [ ] Atualizar o cadastro de cada funcionário na aba "Funcionários" com o
      Gmail correto

  [ ] Definir corretamente o perfil de cada um (Admin ou Funcionário)

  [ ] Garantir que os clientes frequentes têm número de telefone cadastrado
      na aba "Clientes", para que o botão de WhatsApp funcione

  [ ] Informar aos funcionários o link da plataforma e orientá-los a entrar
      com o Gmail cadastrado
