Como usar o Salão Ideal

Este sistema é a agenda digital do salão. Aqui você marca horários, vê quem está agendado e envia confirmações pelo WhatsApp.

O que você vai usar todo dia

1

Ver a agenda de hoje

Clique em Agenda no menu do lado esquerdo.

Os horários aparecem em colunas — uma coluna para cada funcionária.

Use as setas ‹ e › para ir para o dia anterior ou o próximo. O botão Hoje volta para o dia atual.

Quer ver o mês inteiro? Clique no botão Mês no canto superior direito da tela.

-> Você também pode reorganizar as colunas do jeito que preferir — por tipo de serviço, por ordem de chegada, ou como quiser. Passe o mouse sobre o nome de uma funcionária no topo da coluna e arraste para a posição desejada. A ordem fica salva automaticamente.

💡 Reorganizar colunas é uma função exclusiva de Admin. Funcionárias com perfil "Funcionário" não são afetadas.
2

Marcar um agendamento novo

Clique no botão + Agendar no canto superior direito.

Preencha os campos: data, horário, cliente, serviço e funcionária.

Clique em Salvar.

Pronto! O horário já aparece na agenda na hora.

💡 Se o cliente ainda não está cadastrado, vá em Clientes → + Novo Cliente primeiro, depois volte para agendar.
3

Editar ou excluir um agendamento

Clique no bloco colorido na agenda. Um popup abre ali mesmo, sem sair da tela.

No popup você pode alterar diretamente:

  • 📅 Data e horário
  • ✂️ Serviço
  • 💅 Funcionária
  • 🔘 Status (Confirmado / Pendente / Cancelado)
  • 📝 Observações

Clique em Salvar para confirmar. A agenda atualiza na hora.

Para excluir, clique no ícone 🗑️ e confirme. Não tem como desfazer.

💡 Funcionárias com perfil "Funcionário" veem apenas os detalhes do agendamento, sem poder editar.
4

Ver agendamentos sobrepostos

Quando duas ou mais clientes estão agendadas no mesmo horário (ou horários que se cruzam), os blocos ficam empilhados na agenda.

Para ver todos de uma vez, passe o mouse sobre qualquer um dos blocos empilhados — eles se abrem automaticamente em diagonal, como um leque, sem precisar clicar.

Mova o mouse para fora do grupo para fechar o leque.

💡 O agendamento que começa mais tarde fica sempre por cima dos demais, pois é o mais recente.
5

Mandar confirmação pelo WhatsApp

Clique no agendamento na agenda e depois no ícone 📱 dentro do popup.

Escolha o tipo de mensagem: confirmação, lembrete ou cancelamento.

Edite o texto se quiser mudar alguma coisa.

Clique em Abrir WhatsApp — ele vai abrir com a mensagem já escrita, só precisar enviar.

💡 O WhatsApp abre com o número do cliente automaticamente. Não precisa digitar nada.

Gerenciamento Somente Admin

👩‍👩‍👧

Clientes

Menu lateral → Clientes

Cadastre o nome e telefone de cada cliente. O telefone é usado para enviar o WhatsApp.

✂️

Serviços

Menu lateral → Serviços

Cada serviço precisa ter nome, duração em minutos e preço.

💅

Funcionárias

Menu lateral → Funcionários

Use o e-mail Google certinho — é com esse e-mail que a funcionária vai entrar no sistema.

Admin vê tudo e pode criar/editar/excluir.
Funcionário vê só a própria agenda, sem poder alterar nada.

Perguntas frequentes

A agenda não está carregando ou está desatualizada
Clique no botão ↻ no canto superior direito da tela. A agenda também se atualiza sozinha a cada 60 segundos.
Não consigo entrar no sistema
Seu e-mail precisa estar cadastrado na lista de funcionárias pelo Admin. Peça para o Admin verificar.
Só vejo os meus agendamentos, não os das outras funcionárias
Isso é normal quando o seu perfil é "Funcionário". Para ver tudo, o Admin precisa mudar seu perfil para "Admin".
Marquei um agendamento mas ele não apareceu na agenda
Clique no ↻ para atualizar. Se ainda não aparecer, verifique se a data está correta.
Como excluo um agendamento?
Clique no bloco colorido na agenda → no popup que abrir, clique no ícone de lixeira 🗑️ → confirme clicando em "Excluir". Atenção: não tem como desfazer.
Apareceu um aviso amarelo de conflito no topo da agenda
Isso significa que dois agendamentos foram marcados no mesmo horário para a mesma funcionária. Passe o mouse sobre os blocos sobrepostos para ver o leque — clique no que precisar corrigir e altere o horário direto no popup.
Tem vários blocos empilhados e não consigo clicar no que quero
Passe o mouse sobre o grupo — os blocos se abrem em diagonal automaticamente. Clique no bloco desejado enquanto o leque estiver aberto.
